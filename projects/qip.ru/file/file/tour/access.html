Знакомство с возможностями QIP File

Регистрация → Загрузка → Общий доступ → Скачивание → Поиск → Подписки

ШАГ 3 — Обменивайтесь файлами с другими пользователями

Вы можете использовать сервис QIP Sync для личного хранения файлов, чтобы иметь к ним доступ с любого компьютера, подключённого к сети Интернет. Но если Вы хотите поделиться некоторыми Вашими файлами или папками с другими пользователями сети Интернет, QIP Sync может помочь Вам в этом.

  1. Вы можете осуществлять обмен файлами с кем угодно, включая незарегистрированных пользователей сервиса. Сразу после того, как Ваши файлы были загружены на один из серверов QIP Sync, Вы можете поделиться ссылками на эти файлы с другими пользователями сервиса. Чтобы получить ссылку на файл, нажмите на название файла и скопируйте ссылку из поля "ссылка для скачивания" на странице, описывающей свойства файла. Вы можете отправить ссылку по электронной почте, разместить её на веб-странице или опубликовать в любом другом месте в сети Интернет.
  2. Вы можете отправить ссылку на файл другим по электронной почте. Сначала выберите файлы, ссылки на которые вы хотели бы отправить. Затем нажмите кнопку «Отправить ссылку» на панели инструментов Вашего аккаунта. Введите электронный адрес в поле «Получатель:», но, пожалуйста, помните, что вы можете ввести в это поле не более одного электронного адреса. Заполните поле «Тема» и наберите текст сообщения, если нужно. Нажмите кнопку «Отправить» в нижнем правом углу страницы, чтобы отправить ваше письмо. Ссылки на выбранные файлы будут добавлены к Вашему сообщению.
  3. Чтобы открыть общий доступ к Вашей папке, Вам необходима ссылка на папку. Люди, которые будут знать эту ссылку, смогут зайти в эту папку. Самый простой способ получить эту ссылку – открыть эту папку и нажать кнопку "Активировать" рядом со ссылкой на панели инструментов.Чтобы открыть общий доступ к Вашей папке, Вам необходима ссылка на папку. Люди, которые будут знать эту ссылку, смогут зайти в эту папку. Самый простой способ получить эту ссылку – открыть эту папку и нажать кнопку «Активировать» рядом со ссылкой на панели инструментов.
  4. Когда Вы предоставляете другим доступ к Вашей папке, Вы можете настроить различные права и возможности доступа к файлам этой папки для разных людей. Нажмите кнопку «Открыть общий доступ к данной папке» на панели инструментов Вашего аккаунта. В открывшемся окне выберите опцию «Открыть общий доступ к этой папке» и выберите один из возможных вариантов доступа.
  • Загружать файлы – Пользователи смогут самостоятельно загружать файлы в эту папку;
  • Запретить загрузку файлов незарегистрированным пользователям – Только зарегистрированные пользователи 4shared смогут загружать файлы в Вашу общедоступную папку с помощью ссылки на эту папку;
  • Уведомить меня, когда будет закачан новый файл – Отметив эту опцию, Вы получите уведомление по электронной почте, если пользователь загрузит новый файл в общедоступную папку (*);
  • Загрузка из интернета – Пользователи будут иметь возможность загружать файлы из интернета в свою общую папку используя прямую ссылку на файл (*);
  • Показывать только «публичные» файлы – Пользователи могут видеть только файлы со статусом «публичный»;
  • Показывать все файлы – Пользователи могут видеть все файлы в Вашей папке;
  • Просматривать подпапки – Если папка содержит вложенные папки, пользователи смогут видеть эти папки;
  • Создавать вложенные папки – Пользователи будут иметь возможность создавать вложенные папки в вашей общедоступной папке;
  • Переименовывать и удалять файлы – Пользователи могут переименовывать и удалять файлы в данной папке;
  • Права модератора – Пользователи смогут пользоваться всеми функциями, доступными для данной папки;
  • Обновление файлов – Пользователи будут иметь возможность обновить существующие файлы в Вашей общедоступной папке;
  • Загрузка одним нажатием кнопки – Пользователи получают возможность скачать файлы одним нажатием кнопки (*);
  • Разрешить пользователям осуществлять поиск в этой папке – Пользователи будут иметь возможность искать файлы в Вашей общедоступной папке, просто введя имя файла (*).
  • (*) Эти функции доступны только для подписчиков Премиум плана.
  • Вы можете защитить общедоступную папку от несанкционированного доступа с помощью пароля. Чтобы защитить папку с помощью пароля, откройте её и нажмите кнопку «Открыть общий доступ к данной папке» на панели инструментов. Выберите опцию «Открыть общий доступ к этой папке». Отметьте чек-бокс «Добавить пароль» в разделе «Сетевой общий доступ и безопасность» и введите пароль в поля «Пароль» и «Подтверждение пароля». Заметьте, что оба поля чувствительны к регистру, поэтому Вам нужно ввести в них один и тот же набор символов. Нажмите кнопку «Сохранить» для завершения установки пароля. Сообщите пароль только тем пользователям, которым Вы доверяете право доступа к папке.

Вы можете использовать в качестве ссылки на общедоступную папку ссылку более простого и удобного вида, например, myFiles.QIPSync.com с помощью функции «Субдомен». Это функциональное решение доступно на странице настройки доступа к папке, которая открывается после нажатия кнопки «Открыть общий доступ к данной папке».

Подсказка. Если Вы не хотите, чтобы Ваш файл был доступен для всех, Вы должны поделиться ссылкой конфиденциально, только с определенными людьми. Так как ссылка создается секретно, получить доступ к файлу, не зная ссылки, невозможно.
